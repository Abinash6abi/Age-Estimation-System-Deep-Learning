# Age Estimation System Using Deep Learning

An advanced Academic Project developed in MATLAB to estimate human age using specialized image preprocessing and deep learning algorithms.

## Key Features
* **Proposed Face Extraction:** Resizes and converts depth and color images into gray-level representations.
* **Noise Reduction:** Automatically removes black pixels and filters background noise.
* **Ellipse Fitting & Masking:** Utilizes centroid, orientation, and axis length for accurate facial region isolation.
* **Feature Extraction:** Applies entropy calculation on nose color and depth images to capture skin texture complexity.

## Technical Skills Used
* **Language:** MATLAB
* **Domains:** Image Preprocessing, Feature Extraction, Computer Vision
